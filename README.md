# 🐶🐱 Task 03: Cat vs Dog Image Classification using SVM

## 📌 Overview

This project is part of my **Machine Learning Internship at SkillCraft Technology (Task 03)**.
The goal is to classify images of cats and dogs using a **Support Vector Machine (SVM)** model.

---

## 🚀 Project Workflow

* Data loading from Kaggle dataset
* Image preprocessing (resizing, flattening, normalization)
* Feature extraction
* Model training using SVM
* Model evaluation using accuracy and classification metrics
* Prediction on new images

---

## 🧠 Technologies Used

* Python
* NumPy
* OpenCV / PIL
* Scikit-learn
* Matplotlib

---

## 📊 Results

The model successfully classifies images into **cat or dog categories** with good accuracy, demonstrating the effectiveness of SVM in image classification tasks.

* Understanding image preprocessing techniques
* Applying SVM for classification problems
* Handling high-dimensional data
* Evaluating model performance
